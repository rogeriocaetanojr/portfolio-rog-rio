PINEAPP: Foram desenvolvidos protótipos de telas para um sistema de recrutamento e seleção de profissionais focado nas áreas de tecnologia, atendendo diversos requisitos funcionais e não funcionais definidos pelo cliente, uma software house, dando enfoque na experiência e interface de usuário (UI/UX), utilizando Figma para a criação dos protótipos.

HIROGAMA: O projeto foi desenvolvido para a Valmet, empresa multinacional do ramo de energia, celulose e papel. A empresa trouxe o objetivo de incorporar orçamentos aprovados pelos clientes ao Workload, dashboard do planejamento de produção. Utilizando a Power Plataform, desenvolvi um aplicativo em Power Apps, que se integrava dinamicamente ao Power BI, buscando aplicar ferramentas de Data Analytics ao projeto.

Sistema de Reporte de Produção: Foi desenvolvido um sistema para gerenciar reportes de produção para uma indústria de pequeno porte do ramo automotivo utilizando OutSystems. O projeto incluiu a criação de uma interface gráfica e a implementação de funcionalidades de CRUD (Create, Read, Update, Delete).
